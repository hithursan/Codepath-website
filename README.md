# CodePath

CodePath is a responsive static website built to help beginners explore programming languages and development fields, all in one organized place. It was developed as an individual assignment for **Internet and Web Technologies (ICT 142-3)**, Uva Wellassa University of Sri Lanka.

## Live Preview

Open `index.html` in any browser, or enable GitHub Pages for this repository to view it online.

## Features

- **Home page** with hero section, category grid, languages grid, and about section
- **12 development categories** — Web Frontend, Web Backend, Mobile, Desktop, Business Intelligence, Game Dev, AI, Embedded/IoT, Automation, Database, Data Science, Cyber Security
- **25+ programming language pages** — HTML, CSS, JavaScript, TypeScript, React, Java, Python, PHP, C#, Go, Node.js, Kotlin, Swift, Dart, C, C++, Rust, Assembly, Lua, R, Julia, MicroPython, Bash, PowerShell, SQL
- **Light/Dark theme toggle** with preference saved in localStorage
- **Responsive design** across 5 breakpoints (desktop, tablet, and mobile)
- **Hamburger navigation menu** for smaller screens
- **Smooth scroll & scroll-spy navigation** using IntersectionObserver
- **Animated binary background** and gradient accents for a developer-tool feel

## Technologies Used

| Technology | Purpose |
|------------|---------|
| HTML5 | Page structure across all pages |
| CSS3 | Styling, responsive layout, theming with CSS variables |
| JavaScript | Theme toggle, navigation, scroll effects, interactivity |
| Font Awesome | Icons |
| Google Fonts | Typography (Cossette Texte) |
